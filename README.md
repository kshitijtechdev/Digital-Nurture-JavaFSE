## 📌 About The Repository

This repository contains a comprehensive collection of hands-on assignments, practical exercises, domain case studies, and engineering solutions completed as part of the **Cognizant Digital Nurture 5.0 — Java Full Stack Engineering (FSE)** program.

The curriculum is structured to build production-ready full-stack development competency, covering core Java programming, object-oriented software design, database persistence, and modern web application development.

---

## 🛠️ Tech Stack & Core Competencies

| Domain | Technologies & Frameworks |
| :--- | :--- |
| **Programming Language** | Java (JDK 17+), JavaScript (ES6+) |
| **Frontend Development** | HTML5, CSS3, DOM Manipulation, Responsive UI |
| **Backend Architecture** | Core Java, OOPs Design, Collections Framework, Exception Handling, Multithreading |
| **Data Persistence** | SQL, MySQL, Relational Database Design, JDBC Connectivity |
| **Tooling & Workflow** | Git, GitHub, VS Code, Eclipse / IntelliJ IDEA |
